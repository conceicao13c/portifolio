Portfólio — Conceição Ramos

Bem-vindo(a) ao meu portfólio.
Este projeto está sendo desenvolvido para apresentar minha trajetória, habilidades e projetos enquanto avanço na área de Desenvolvimento Front-End.

O site foi construído utilizando HTML5, CSS3 e JavaScript puro, com foco em responsividade, desempenho e elementos visuais dinâmicos.

Funcionalidades
Layout Responsivo

Utilização de <picture> para fornecer imagens diferentes conforme a largura da tela.

Estrutura adaptada para dispositivos móveis e tablets.

Animação Dinâmica dos Peixes

O arquivo script.js implementa animações com:

Movimentação aleatória contínua

Mudanças naturais de direção

Detecção de proximidade do mouse com reação de “fuga”

Colisão com bordas da viewport

Execução otimizada via requestAnimationFrame

Animações de Entrada

Componentes como navbar, texto principal, imagem e rodapé possuem animações suaves utilizando keyframes CSS (zoom e fade).

Estilo Visual

Background com gradiente

Tipografia usando Google Fonts (Oswald)

Layout minimalista com foco na apresentação de forma clara
